Mission

Our mission is to develop practical educational tools that can be used to help students in topics that traditionally can prove difficult in interesting, interactive, and intuitive ways!

Project Description

We're creating a website with graphical visualization tools to assist young students in learning mathematical concepts such as different types of triangles and fractions. All of the visualization tools are written in Javascript using the Processing library, while the website is written using HTML/CSS with the Semantic UI framework. The fraction tool helps graphically explain the numerator and denominator of a fraction to students, and the triangle tool lets students visually see the difference between a right, obtuse, and acute triangle.
